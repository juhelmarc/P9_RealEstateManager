# RealEstateManager
OpenClassrooms Project 9 : Android mobile application allowing agents to access different real estate listings from their smartphones or tablets.

**Skills**

- Material Design
- Google Map
- SQLite Room
- MVVM
- RecyclerView
- LiveData
- Coroutine Flow
- ViewBinding
- Unit test with Mockito
- Dagger Hilt (dependency injection)
- Image Slider
- Coil
